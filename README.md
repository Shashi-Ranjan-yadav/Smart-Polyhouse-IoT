# Smart Polyhouse Automation

This project is an IoT based smart polyhouse monitoring system.

## Features
- Temperature monitoring
- Humidity monitoring
- Soil moisture monitoring
- Automatic irrigation
- Fire detection
- Mobile notification using Blynk

## Hardware Used
- ESP32
- DHT11 sensor
- Soil moisture sensor
- Flame sensor
- Relay module
- LCD display

## Software Used
- Arduino IDE
- Blynk IoT
- GitHub
